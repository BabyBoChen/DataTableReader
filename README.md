# 資料表轉換器

「資料表轉換器」是一款使用.Net的Windows Presentation Foundation(WPF)框架所開發而成的桌面應用程式。「資料表轉換器」可以對資料庫執行SQL指令，並將查詢結果儲存成XML檔或是JSON檔。「資料表轉換器」支援的資料庫類型包含PostgreSQL、SQL Server和SQLite。<br/>

# 系統需求
作業系統：Windows 10<br/>
執行環境：.NET Desktop Runtime ^5.0.16<br/>
